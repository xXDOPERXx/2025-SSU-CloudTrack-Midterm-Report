# Hybrid Cloud란
[[Public Cloud]] + [[Private Cloud]] 형태의 클라우드 모델
## 특징
**제어** : 사용자가 원하는 서비스와 데이터를 직접 관리할지, [[Cloud Service Provider|CSP]]에게 맡길지 직접 제어 가능
**유연성** : [[Public Cloud]]에서 리소스를 유연하게 사용하여 서비스를 제공하는 것이 가능
**비용 효율성** : [[Public Cloud]]와 [[Private Cloud]]에 서비스의 리소스 비율을 조정하여 효율적인 비용 관리 가능 
**용이성** : [[Public Cloud]]와 [[Private Cloud]]r간 마이그레이션 전환의 용이함

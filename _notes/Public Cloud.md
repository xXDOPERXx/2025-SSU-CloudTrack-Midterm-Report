# Public Cloud란
[[Cloud Service Provider|CSP]]가 구축한 서버, 스토리지 등의 IT 인프라를 사용료를 내고 사용하는 형태의 [[Cloud|클라우드]] 모델
## 특징
**비용 절감** : 소유할 장비나 소프트웨어 없이 사용한 서비스에 대한 사용료만 지불
**유지 관리 X** : 관리할 장비나 소프트웨어 X, [[Cloud Service Provider|CSP]]가 유지 및 관리함
**높은 안정성** : 방대한 서버 네트워크와 리소스로 실패를 방지
**높은 [[Scalability|확장성]]** : 사용자의 요구 사항은 [[Cloud Service Provider|CSP]]가 제공하는 상품형 리소스 형태로 완벽에 가깝게 처리할 수 있음

## SaaS
[[Cloud Service Provider|CSP]]가 모든 수준의 서비스를 제공하고, 사용자는 어플리케이션을 사용하는 형태의 [[Cloud|클라우드]] 모델
![](/assets/img/스크린샷 2025-10-30 오전 11.05.44.png)

aka. 주문형 SW or 서비스형 SW
사용자는 필요한 기능을 필요한 만큼 이용하고 사용료를 지불하는 형태
**사용자**는 책임 범위 없이 사용만 한다는 점에 주목
### SaaS를 제공하는 기업
![](/assets/img/Pasted image 20251030111617.png)
### SaaS의 조건
- Configuration
- Multi-Tenancy
- [[Scalability]]
### SaaS 성숙도 모델
![](/assets/img/스크린샷 2025-10-30 오전 11.48.40.png)
**Level 1** : Custom Tenant
- provides service as (sets of 1 client - 1 app instance)
**Level 2** : Single Tenant
- provides service as (1 client - 1 app instance)
- Configuration 
**Level 3** : Multi Tanent
- provides service as (multi client - 1 app instance)
**Level 4** : Scalable Multi Tenant
- provides service as (multi client - scalable 1 app instance)
## 특징
- 별도 설치 없이 인터넷만으로 가능
- 사용한 만큼 청구되는 구독형 서비스
- 보안성이 높고 접근이 자유로움
- 유지 관리에 소모되는 리소스와 비용 X
